本MIB遵循的标准：
本MIB属于我司定义的私有MIB。

在本项目中支持哪些版本：
在本项目（10.4项目）中，10.4(1)以及之后的版本都支持。

在这些版本中支持哪些产品：
10.4(1)以及之后的版本，其对应的所有产品都支持。

MIB对象实现程度：
MIB表格对象ruijieIfAvailableBWTable的元素支持只读；
MIB表格对象ruijieIfTable中的元素ruijieIfEntryStatus只支持只读（该对象在定义文件中为可读创建）；
MIB表格对象ruijieIfIpTable未实现；
MIB表格对象ruijieIfStatusTable未实现；
MIB表格对象ruijieGlobalIfDisableRecovery未实现；
MIB表格对象ruijiePortTypeChooseTable未实现；
MIB表格对象ruijieIfMTUTable未实现；
MIB表格对象ruijieInterfaceTraps未实现；
其他的实现与MIB定义文件一致。
